# 🥁 JavaScript Drum Kit

An interactive drum kit web application built using **HTML, CSS, and JavaScript** that allows users to play different drum sounds using their keyboard keys. This project is part of the **JavaScript30 Challenge**.

## 🚀 Live Demo

👉 [https://taseen2.github.io/Drum-roll/](https://taseen2.github.io/Drum-roll/)

---

## 📌 Features

* 🎹 Play drum sounds using keyboard keys
* 🔊 Realistic audio effects for each drum key
* ✨ Smooth key press animations
* ⚡ Instant response with JavaScript event listeners
* 🎨 Stylish and responsive UI
---

## 🛠️ Technologies Used

* **HTML5** – Structure of the drum kit
* **CSS3** – Styling, layout, and key animations
* **JavaScript (ES6)** – Event listeners, audio playback, DOM manipulation

---

## 🎮 How to Play

1. Open the live website link above

2. Press the corresponding keyboard keys shown on screen:

   * A
   * S
   * D
   * F
   * ;
   * L
   * K
   * J

3. Each key triggers a unique drum sound

## 📚 Concepts Practiced

* `addEventListener()`
* `keydown` event
* `querySelector()`
* `data-key` attributes
* Audio object control
* `currentTime` reset for rapid replay
* CSS `transform` and `transition`
